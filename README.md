# Alter

The ALTERnative music streaming application. (backend)

## How to build

- Install Go from your package manager.
- Clone this repository:

    ```sh
    git clone https://github.com/GuillaumeMZ/alter-backend.git
    ```

- Navigate to the repository:

    ```sh
    cd alter-backend
    ```

- Build the project:

    ```sh
    go build
    ```

    The generated executable is named `alter-backend` and is in the root folder.
